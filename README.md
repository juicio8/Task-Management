## Getting Started
1. Clone this repository.

## Installation
1. Ensure you have node.js installed.
2. Make sure you are in this repo's directory.
3. Run <code>npm install</code>

## Run Project
1. Run <code>npm run dev</code>
2. Go to <code>http://localhost:5173/</code>
